== Description ==

Run an audio player RADIO87 inserted into WordPress posts and pages.

To activate the player, just write " [RADIO87PL] "


== Installation ==

1. Download plugin-wp-radio87.zip from http://www.radio87.fr/docs/plugin-wp-radio87.zip (direct download link)

2. Decompress the file contents

3. Upload the plugin-wp-radio87 folder to your WordPress plugins directory (/wp-content/plugins/)

4. Activate the plugin from the WordPress Dashboard

5. See instructions for using the plugin

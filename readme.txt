== Description ==

Run an audio player RADIO87 inserted into WordPress posts and pages.

The automatic player between flash and html5, just write " [RADIO87PLAUTO] "

To activate the flash player, just write " [RADIO87PL] "

The flash player without autoplay : " [RADIO87PLAD] "

To activate the html5 player, just write " [RADIO87PL2] "

The html5 player without autoplay (recommended for ANDROID) : " [RADIO87PL2AD] "

To display a link to the external player : " [RADIO87PLEXT] "

To display the current title from RADIO87 : " [RADIO87CT] "

To display a link to the tracklist from RADIO87 : " [RADIO87TR] "

To display the local time of RADIO87 : " [RADIO87HR] "

To display a link to Facebook : " [RADIO87FACEBOOK] " – Twitter : " [RADIO87TWITTER] " – Tumblr : " [RADIO87TUMBLR] " – Weibo : " [RADIO87WEIBO] " – Messager : " [RADIO87MESSAGER] " – Twitter #RADIO87 : " [RADIO87HASHTAGTWITTER] " – Twitter #SuivezlaTortue : " [SUIVEZLATORTUEHASHTAGTWITTER] " – Chat room : " [RADIO87CHAT] "



== Installation ==

1. Download plugin-wp-radio87.zip from http://www.radio87.fr/docs/plugin-wp-radio87.zip (direct download link)

2. Decompress the file contents

3. Upload the plugin-wp-radio87 folder to your WordPress plugins directory (/wp-content/plugins/)

4. Activate the plugin from the WordPress Dashboard

5. See instructions for using the plugin
